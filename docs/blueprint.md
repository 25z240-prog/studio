# **App Name**: Hostel Chow Vote

## Core Features:

- Menu Item Submission: Allow hostel residents to propose new food menu items, ensuring a dynamic and community-driven selection.
- Menu Item Display: Show the current proposed menu items in a scrollable list with title, ingredients, photo, preparation instructions, nutritional facts and dietary information, such as vegan or vegetarian
- Chow Rank Voting: Enable hostel residents to vote on their favorite proposed menu items through a ranking system, indicating preferences.
- Realtime results tallying: Sum up and update the latest vote totals on each menu item

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5), evoking trust and a sense of community.
- Background color: Light Gray (#EEEEEE), to ensure content and votes pop.
- Accent color: Vibrant Orange (#FF5722) for interactive elements like vote buttons.
- Body font: 'PT Sans', a modern and welcoming sans-serif for readable content.
- Headline font: 'Space Grotesk' to convey contemporary community and scientific data gathering
- Use food-related icons in a flat design style to represent menu categories and actions (voting).
- Subtle animations and transitions on vote submissions to enhance user experience.
